上虞地处杭州湾南翼，是融入杭州湾大湾区的前沿阵地，也是绍兴接轨大上海的桥头堡。区域面积1403平方公里，拥有一个国家级开发区、一个省级旅游度假区、两个省级特色小镇，形成了机械装备、医药化工、轻工纺织、照明电器、光伏及绿色能源五大主导产业，境内主板上市公司16家，全国民营企业500强3家，规上企业700余家。2018年实现地区生产总值916亿元，财政收入141.25亿元，全国综合实力百强区排名第40位。

![IMAGE](/img/Post_2019_07_03_IMG_01.png)

<!--more-->
为更好地帮助海内外人才实现创业梦想，北京海创智库科技服务中心整合全球科技服务资源，联合上虞区共同举办2019“海创人才”国际创业大赛，通过“以赛代引”的方式，为参赛者提供政策、产业、资本、空间、市场等全方位的支持和保障，吸引更多人才落户上虞，推动地区创新发展。

# 一、主题

海聚英才·创汇娥江

# 二、时间（以下所渉均为北京时间）

启动时间2019年6月，决赛时间11月中旬

# 三、组织架构

（一）指导单位

浙江省外国专家局  中共绍兴市委组织部

（二）主办单位

中共绍兴市上虞区委、绍兴市上虞区人民政府

（三）承办单位

中共绍兴市上虞区委人才工作领导小组办公室

北京海创智库科技服务中心

陕西光电子集成电路先导技术研究院

（四）协办单位

全球仟寻-杭州仟寻人力资源有限公司

# 四、参赛对象及条件

（一）人才要求：所有海内外高层次人才，不超过50周岁（特别优秀者可以适当放宽），具有国内博士学位或海外硕士及以上学位，3年及以上海外工作经验，社会信誉良好，无不良记录。

（二）领域要求：电子信息、人工智能、大数据、互联网相关、高端装备制造、金属加工、水处理、新材料、现代医药、高端纺织、绿色化工、节能环保、半导体等相关产业领域具备原始创新能力和成果转化能力的项目或团队。

（三）参赛项目拥有自主知识产权或发明专利，知识产权（发明专利）权属明晰，参赛项目与知识产权直接具有相关性。

（四）参赛核心团队结构完整，由生产管理、市场开发、技术研发等方面人才组成，整体综合实力较强；经营模式科学合理，发展目标清晰。

（五）参赛团队报名阶段尚未以参赛项目在绍兴注册成立企业，有意向于赛后12个月内在上虞注册落户。

# 五、赛程安排及说明

（一） 赛程安排

1. 报名：2019年6月至8月

2. 启动仪式&上海城市赛（数字经济）：7月13日

3. 深圳城市赛（智能装备）：8月3日

4. 西安城市赛（半导体、智能制造）：8月10日

5. 北京城市赛（新材料）：8月24日

6. 考察、对接和尽调：7月-10月

7. 上虞决赛：11月中旬

（二） 赛程说明

1. 报名截止时间：

（1）数字经济领域：7月1日

（2）高端制造领域：7月15日

（3）半导体、智能制造领域：7月20日

（4）新材料领域：8月5日

2. 项目初筛：由海创人才团队，邀请对口领域专家进行评审，初筛结果通过大赛网页及时发布。

3. 7月-10月，每场城市赛后，组织拟入选“娥江英才”计划的参赛选手赴上虞进行实地考察（以下简称“考察”），不参加考察环节的选手视为自动放弃比赛资格。海创人才科技经纪团队和上虞区组织部为参赛选手提供政策解析、答疑解惑、落地意向对接服务。

4. 城市赛现场路演采取“8+7”的模式进行（即8分钟PPT演示+7分钟互动问答）。评委由专业领域内2位技术专家和3位投资人组成。10月中旬在大赛网页公布入围决赛名单。

5. 决赛阶段：按照“8+10”的模式进行现场路演（即8分钟PPT演示+10分钟互动问答）。评委组由技术专家、资深投资人及企业家按照一定比例组成。

# 六、奖项设置和扶持政策

（一）奖项设置

依据入围决赛项目质量和数量设立一、二、三等奖和优胜奖若干名。

奖金额度（税前）为**一等奖30万元**；**二等奖20万元**；**三等奖10万元**；**优胜奖5万元**。奖金分为2次拨付，大赛结束及项目落地后各拨付50%，个税自理。

（二）扶持政策

城市赛中获评的**A类**、**B类**、**C类**和**D类**项目，经尽职调查后12个月内落地上虞的，可分别获得**500万元**、**300万元**、**200万元**、**100万元**的创业资助支持。

落地上虞的创业项目可享受三年免租办公场所和部分中试场房的租金补助。给予贷款额1000万元内基准利率两年全额贴息，销售、研发、税收、购房、租房、子女入学、疗休养等方面的优惠和支持政策。

（三）交通补贴

交通补贴发放标准为：海外人才6000元/项目，国内省外人才3000元/项目，省内人才1000元/项目，考察交补1500元/项目。海外人才须提供参赛日前一周内的回国机票和入境签证页复印件，国内人才须提供参赛日前两日内赴比赛地各类交通工具票据复印件。参赛交补待每场比赛结束后统一发放；考察交补待考察上虞之后再行发放。

# 七、补充说明

1. 本次大赛不对参赛选手收取任何费用。

2. 参赛选手不得违反大赛规定，所提交的创业项目相关材料不得存在抄袭、盗用等不法现象，一经发现，即取消申报及资助资格，追回已资助资金。涉及参赛项目产权及相关法律纠纷的，由参赛选手自行承担一切责任。

3. 大赛报名项目材料仅用于项目评审及资源对接，城市赛和决赛过程将对社会公开。

4. 参赛选手完全本着自愿原则参加本次比赛，如对大赛有异议，可及时联系沟通，或选择退出比赛，但无权要求更改比赛规程或赔偿因参赛所发生的的任何费用。

5. 因不可抗力因素影像大赛正常进行的，主办方有权对赛程作相应调整或终止比赛。

6. 大赛组委会保留进一步补充本次比赛规程的意见和最终解释权。未尽事宜，另行通知。
