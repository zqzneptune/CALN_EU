# 大赛背景

为贯彻落实相城区人才引进三年计划（2018-2020年），紧紧围绕“大研发、大文化、大健康”三大产业定位，打造全国领先的大数据、工业互联网、科技金融、智能驾驶、先进材料、生物医药“六大未来产业高地”，以“人才带项目、项目配基金”模式积极向海内外宣传推介相城区创新创业环境和人才政策，大力招引全球优质人才、项目落地发展，促进人才、技术和资本有机结合。根据2019年全区人才工作要点和“双招双引”工作安排，2019年（第十届）阳澄湖创客大赛实施方案建议如下：

# 组织机构

1.主办单位

中共相城区委、相城区人民政府

2.承办单位

相城区人才办、人社局、工信局、科技局、商务局

3.协办单位

苏州市相城金融控股（集团）有限公司

4.支持单位

**荷兰华人生命科学协会 (苏州市离岸孵化基地， 苏州相城区海外引才联络站)**

# 参赛条件

参赛人才(团队)必须符合以下条件：

1.硕士研究生及以上学历学位，年龄50周岁以下(1969年1月1日以后出生)，身体健康，无不良信用和违法违纪记录。

2.参赛项目应符合相城大数据、工业互联网、科技金融、智能驾驶、先进材料、生物医药六大未来产业定位，有较高创新水平和较强市场竞争力、有较好的潜在经济效益和社会效益、易于形成新兴产业的高新技术成果且具有核心技术、完全知识产权、高附加值的创新项目。

3.2019年7月1日前尚未在苏州市相城区注册成立企业的且有意向最迟于赛后1年内在相城区落户。

4.已在苏州大市范围内获得人才计划项目支持的人才(团队)不得参加大赛。

# 赛程安排

## 1.荷兰分站赛

2019年（第十届）阳澄湖创客大赛将于珠三角、以色列、长三角、荷兰分别组织分站赛。组织专家对各分站赛报名项目进行初选，遴选10个左右项目晋级分站赛，通过现场路演答辩的形式评选出6名获奖选手入围总决赛项目评审。

荷兰分站赛：将于9月下旬在荷兰阿姆斯特丹举办分站赛，采用跨境路演形式。

## 2.入围选拔

组织专家对分站赛获奖项目进行网络评审，遴选优质项目入围总决赛。

## 3. 总决赛

将于苏州相城经贸恳谈会活动期间，举办2019年（第十届）阳澄湖创客大赛

1.项目评审：组织专家对各分站赛及线上选拔选出的项目进行决赛项目评审，15名左右选手晋级总决赛。

2.现场比赛：采取现场路演答辩的形式，每个决赛项目路演6分钟、问答4分钟。邀请7名创投专家、创业导师、技术专家组成评审团，对总决赛选手进行现场提问和打分，现场打分结束后即时公布获奖名单。

3.实地考察：邀请获奖选手参加苏州相城经贸恳谈会活动的开幕式及现场颁奖仪式，并组织选手参加实地考察和对接洽谈。

(二)扶持政策

1.总决赛前6名获奖选手一年内在相城注册公司并实体化运作，且所创办企业实收资本中的现金出资不低于50万元(实收资本中现金出资占企业注册资本比例不低于30%或选手为自然人第一大股东、持股比例不低于20%)，可申报公司注册后的最新一批申报阳澄湖科技领军人才计划(大赛获奖类)，通过形式审查后直接进入实地考察环节。

2.总决赛获奖项目可优先入驻区内科技企业孵化器，并享受园内基地的优惠措施和服务。

3.统筹协调人社、科技，市场监管(工商)、税务等部门，为人才落户、项目落地提供一站式、一条龙服务，并积极协助落户人才(团队)申报各类高层次人才计划。

4.决赛获奖项目，可获得相城区级各类基金累计最高700万元股权融资额度，包括：

1）相城区天使投资引导基金提供的最高200万元的天使投资；

2）相城区创投基金、产业发展基金提供的不超过500万元的股权投资。
